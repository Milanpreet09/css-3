description of html tags

  • !doctype html : this tag is used to describe the verson of the html document.
  • br tag:this tag is used to provide a break means the text will be desplayed on the next line after using this tag.
  • div tag: it is used to define the section in the document.
  • a:this is a anchor tag that is used to create a link or hyperlink.
  • legend:it is a tag that is used to define a caption for the content of fieldset.
